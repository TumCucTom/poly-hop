# 🎮 Poly-Hop: Game Summary

## ✅ What We Built

A fully functional 2D retro platformer game controlled by webcam body tracking!

### 🎯 Core Features
- **Webcam Body Tracking**: Jump, lean left/right, duck using MediaPipe
- **Retro Pixel Art**: 16x24 pixel customizable character
- **Platformer Gameplay**: Jump, run, collect coins, defeat enemies
- **Character Customization**: Skin tone, outfit, hair style options
- **Real-time Movement Log**: See all detected body movements
- **Procedural Levels**: Randomly generated platforms and enemies

### 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Game Engine**: p5.js for 2D graphics
- **Computer Vision**: MediaPipe Pose detection
- **Backend**: Node.js with Express
- **Styling**: Custom retro pixel art CSS

## 🚀 How to Run

1. **Install dependencies**: `npm install`
2. **Start server**: `./start.sh` or `node server.js`
3. **Open browser**: `http://localhost:3001`
4. **Grant webcam access** and start playing!

## 🎮 Controls
- **Jump**: Raise arms or jump in real life
- **Move Left/Right**: Lean body left or right
- **Duck**: Lower body or crouch down
